== Description ==

Tourily plugin creates custom property listings and real estate virtual tours. This
plugin works with a free [Tourily.com](http://www.tourily.com/) account in order
to provide you with search engine optimized virtual tours on tour own website.

Features in V1.0:

* Search engine optimized virtual tours.
* Property listing detail page.
* Widgets to display listings in widget ready areas.
* Shortcodes to embed property listings within pages or posts.

To use this plugin you need an active username and password at Tourily.com.

== Installation ==

Upload the plugin and activate. In settings enter your Tourily username and passwords. Done.

Listings created on Tourily.com will display on your Wordpress site.
